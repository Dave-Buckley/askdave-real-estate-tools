Inputs (all text fields, controlled state with `useState<string>`):
- Property Price (AED) -- empty default, placeholder "e.g., 2,000,000"
- Commission Rate (%) -- default "2" (from CALCULATOR_RATES.commission.defaultRate)
- Listing Agent Share (%) -- default "50"
- Buyer's Agent Share (%) -- auto-calculated as `100 - listingPercent`, editable (if agent edits buyer's share, listing share auto-adjusts)

Live calculation via `useMemo` (recalculates on every input change):
- Total Commission = propertyPrice * (commissionRate / 100)
- Listing Agent Share (AED) = totalCommission * (listingPercent / 100)
- Buyer's Agent Share (AED) = totalCommission * (buyerPercent / 100)
- VAT on Listing Agent = listingShare * 0.05
- VAT on Buyer's Agent = buyerShare * 0.05
- Total incl. VAT = totalCommission + totalVAT

Guard: if propertyPrice is 0 or empty, return null results. Show empty state placeholder "Enter property price to calculate".

Results display (only when valid inputs):
- "Total Commission" line: formatAED amount
- Divider

- "Buyer's Agent (XX%)" line: formatAED amount, then below it "VAT (5%): formatAED" in smaller muted text
- Divider
- "Total incl. VAT" line: formatAEDDecimal amount, bold
