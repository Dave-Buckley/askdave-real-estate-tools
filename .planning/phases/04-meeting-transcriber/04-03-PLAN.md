1. **Create `src/renderer/panel/components/RecorderOverlay.tsx`** -- The UI that renders inside the pop-out recorder window (280x120px). This component handles the actual recording and transcription.

**Component behavior:**
- On mount: get whisper model preference via `window.electronAPI.getWhisperModel()`, create the Whisper Web Worker using Vite pattern: `new Worker(new URL('../workers/whisper-worker.ts', import.meta.url), { type: 'module' })`, send `{ type: 'load', model }` to pre-load the model
- Show model download progress if model is being downloaded (progress bar)
- Once model is ready, show a large "Start Recording" button
- On start: instantiate `AudioRecorder`, call `recorder.start()`, relay state to main window via `window.electronAPI.transcriberStateUpdate({ state: 'recording', elapsed: 0 })`, start an interval timer updating elapsed every second
- Show recording UI: elapsed time (MM:SS format), pulsing red dot, Pause button, Stop button
- On pause: call `recorder.pause()`, relay `{ state: 'paused', elapsed }`, show Resume button instead of Pause
- On resume: call `recorder.resume()`, relay `{ state: 'recording', elapsed }`
- On stop: call `recorder.stop()` to get Float32Array, relay `{ state: 'transcribing' }`, show "Transcribing..." with spinner, send `{ type: 'transcribe', audio }` to worker (use transferable: `[audio.buffer]`)
- On worker result: call `window.electronAPI.transcribeComplete(text)` to relay transcript to panel via main process. The main process will close the pop-out window.
- On worker error: show error message, allow retry or close
- On component unmount or window close: clean up recorder (stop if active), terminate worker

**Styling:**
- Match app dark theme: `bg-[#0d0d0e]`, `text-[#ededee]`
- Custom title bar for drag (same pattern as main TitleBar but minimal -- just a drag area and a close X button)
- Compact layout fitting in 280x120
- Use Lucide icons: `Mic`, `MicOff`, `Pause`, `Play`, `Square` (stop), `X` (close), `Loader2` (spinner)
- Pulsing red dot for recording state: `animate-pulse` with red bg
- Progress bar for model download: simple div with width% and indigo bg

**Max recording warning:** When AudioRecorder's `onMaxReached` fires (60 min cap), auto-stop and begin transcription. Show a brief toast/message "Maximum recording length reached".

**d) Add transcriber state listener (useEffect):**
```typescript
useEffect(() => {
  window.electronAPI.onTranscriberState((state) => {
    if (state.state === 'recording') {
      setIsRecording(true)
    } else if (state.state === 'paused') {
      setIsRecording(false) // stop pulsing, but don't clear recording flag
    } else if (state.state === 'done' && state.transcript) {
      setIsRecording(false)
      setTranscript(state.transcript)
      setView('transcriber')
    } else if (state.state === 'idle' || state.state === 'error') {
      setIsRecording(false)
    }
  })
  return () => {
    window.electronAPI.removeTranscriberStateListener()
  }
}, [])
```
